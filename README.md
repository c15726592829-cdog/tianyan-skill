# Tianyan Skill

> 大道五十，天衍四九，人遁其一。
>
> 天行健，君子以自强不息。

**Tianyan** 是一个面向文王六爻的纳甲排盘与分析 Skill。它以传统卦理观察事物的结构、变化与趋势，却不把卦象视为不可更改的命令。

“四十九”代表可以推演和认识的部分；遁去的“一”，则留给人的智慧、勇气、实力与行动。卦象提供参考，选择仍归于人。所谓自强不息，正是在看见局势之后，仍以自身行动开辟可能。

在工程实现上，Tianyan 使用确定性排盘、可审计的三硬币模拟和明确的分析边界，力求让每一项排盘结果可复核、每一层判断有依据，并始终将现实事实置于象数推演之上。

## 支持范围

- 文王六爻
- 用户明确要求时，由脚本模拟三枚硬币抛六次代起卦
- 纳甲排盘
- 世应、六亲、六神、伏神
- 月建、日辰、旬空
- 动爻、变卦分析

## 不支持范围

- 梅花易数
- 数字起卦
- 时间直接起卦
- 体用法

## 输入格式

```text
问题：
起卦时间：
爻象：
```

示例：

```text
问题：这件事近期是否会有进展？
起卦时间：2026-06-16 15:30
爻象：7 8 9 7 8 6
```

爻象必须自下而上输入六个值：

| 值 | 含义 |
|---|---|
| `6` | 老阴，动 |
| `7` | 少阳，静 |
| `8` | 少阴，静 |
| `9` | 老阳，动 |

也可直接输入：

```text
老阴、少阳、少阴、老阳、少阳、少阴
```

起卦时间必须精确到分钟，例如 `2026-06-16 15:30`。

## 代起卦

用户明确要求“你帮我起卦”时，可以运行：

```bash
python scripts/qi_gua.py --question "问题" --timezone Asia/Shanghai
```

脚本采用传统三硬币法，六次结果自下而上：

| 组合 | 数值 | 爻 |
|---|---:|---|
| 三字 | `6` | 老阴，动 |
| 两字一背 | `7` | 少阳，静 |
| 一字两背 | `8` | 少阴，静 |
| 三背 | `9` | 老阳，动 |

其中字面记 `2`，背面记 `3`。脚本使用操作系统安全随机源，公开原始熵、
18枚硬币结果和六爻值，并以脚本实际执行时刻作为起卦时间。

每次现场起卦只运行一次，不得因结果不满意而重抛。重放参数仅用于核验
已经公开的结果，不得解释成一份新卦。

## 输出顺序

每次完整解读固定采用以下顺序：

1. 排盘结果
2. 卦意解读：本卦、动爻、变卦，只列原文和白话文
3. 纳甲分析：用神、月日、世应、六亲、动变与空破合冲
4. 结论

卦意解读不映射用户、现实人物或所问事项；具体问题的判断从纳甲分析开始。
本卦与变卦先列卦辞、大象原文，动爻先列爻辞原文，再给出白话文。
不得把白话转述标为原文；原文无法核实时必须明确说明。
