# Tianyan Skill

文王六爻纳甲排盘与断卦辅助 skill。

## 支持范围

- 文王六爻
- 纳甲排盘
- 世应、六亲、六神、伏神
- 月建、日辰、旬空
- 动爻、变卦分析

## 不支持范围

- 梅花易数
- 数字起卦
- 时间直接起卦
- 体用法

## 输入格式

```text
问题：
起卦时间：
爻象：
```

示例：

```text
问题：这件事近期是否会有进展？
起卦时间：2026-06-16 15:30
爻象：7 8 9 7 8 6
```

爻象必须自下而上输入六个值：

| 值 | 含义 |
|---|---|
| `6` | 老阴，动 |
| `7` | 少阳，静 |
| `8` | 少阴，静 |
| `9` | 老阳，动 |

也可直接输入：

```text
老阴、少阳、少阴、老阳、少阳、少阴
```

起卦时间必须精确到分钟，例如 `2026-06-16 15:30`。

## 输出顺序

每次完整解读固定采用以下顺序：

1. 排盘结果
2. 卦意解读：本卦、动爻、变卦，只解释原始卦爻含义
3. 纳甲分析：用神、月日、世应、六亲、动变与空破合冲
4. 结论

卦意解读不映射用户或现实人物；具体问题的判断从纳甲分析开始。
