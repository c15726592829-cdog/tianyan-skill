# Objects and Lost Property Module

## Scope

- Category ID: objects-lost-property
- Chinese label: 手持物、物品类别、遗失物。
- Use when identifying an object or locating a lost/hidden item is primary.

## Trigger Questions

- 我的左手拿的什么？
- 找不到的小物件在哪类地方？
- 我包里现在放着什么重要东西？

## Exclusions and Redirects

- If the main answer is direction/place, add or route to `location-direction`.
- If the object is money, debt, or payment, use `money-debt`.
- If the object is a contract, certificate, or procedure file, use `contracts-documents`.

## Question Subject

Distinguish held-object tests from lost-object searches. Held-object questions
ask category identification; lost-object questions ask both object anchor and
place/visibility.

## Perspective

State whether the object is in the asker's hand, possession, room, travel bag,
or unknown environment. Exact modern product identification is high-risk.

## Primary Anchor or Useful Spirit

Use the object/matter line relevant to the item class. For unspecified held
objects, start with direct trigram/body/nature imagery and seek independent
structural support.

## Secondary References

Body correspondences, trigram nature, line relative, branch element, six spirit,
material/function evidence, and candidate set can support object class.

## Anchor Conflict Resolution

If image evidence suggests a class but useful-spirit evidence conflicts, output
the broad class and list alternatives rather than forcing exact item.

## Month and Day

Month/day judge whether a lost object is recoverable, hidden, damaged, moved,
or activated; they do not name the object by themselves.

## Moving and Changed Lines

Movement can show the object being moved, revealed, changing place, or involved
in current action. It does not prove exact product identity.

## Shi and Ying

Shi may mark the asker's possession; Ying may mark outside place or another
person's possession. Use only after fixing the object scenario.

## Branch Structures, Void, Break, Tomb, and Hidden Spirit

Hidden/void/tomb evidence may describe concealed, covered, hard-to-find, or
temporarily unavailable objects.

## Hexagram and Trigram Corroboration

Use `乾为首`, `坎为耳`, `离为目`, `艮为手`, `兑为口` and natural images as object
class support. Exact products need a second independent evidence group.
For color questions, a six spirit directly on the selected object/useful-spirit
line may outrank broad trigram color; broad trigram color may outrank loose
Najia element color. If the selected object line is uncertain, do not force an
exact color.

## Allowed Answer Resolution

Allowed: broad class, material/function category, ranked candidates. Exact item
is at most medium confidence with two independent evidence groups; otherwise weak.

## Required Contrary Evidence

Report one-signal object guesses, lack of candidate set, conflicting images,
weak/hidden anchor, or overbroad category.

## Prohibited Inferences

Do not infer a specific audio device from 坎为耳 alone; do not reverse-fit an
object after the reveal as if it had been proven.

## Worked Structural Examples

- “左手拿什么”：use hand/body imagery plus another line/material/function signal before exact object.
- “小物件在哪”：object class may be known; location-direction module handles place evidence.
- “包里有什么”：without candidate set, answer category and ranked candidates, not guaranteed exact item.
